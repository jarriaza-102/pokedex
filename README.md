# Simple Pokedex

This project is built with React, Typescript & styled-components.
It also has functional tests built with jest & testing library.

This project is a simple Pokedex that list the pokemons from [PokeApi](https://pokeapi.co/), and you can check more details for each Pokemon expanding it, or toggle to favorite and check the favorite pokemons under Favorite tab.  

## How to run it?

In order to run the project, after you cloned the project in your local machine, you'll have to run `npm install`, and aftere that, `npm start`, and that's it! You can poken : `http://localhost:3000/` and check how the app works.

## Run the tests

In order to run the functional tests, you'll have to run: `npm run test -- a`
